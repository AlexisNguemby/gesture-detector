# Hand Tracking with MediaPipe & OpenCV

A real-time hand tracking application using MediaPipe and OpenCV.  
It detects hand landmarks from your webcam and draws a hand skeleton overlay in real time.

## Features

- Real-time webcam hand detection
- Hand landmark tracking (21 points per hand)
- Visual skeleton drawing with connections
- Automatic fallback if video mode is not supported
- Simple exit controls (press `Q` to quit)

## Tech Stack

- Python
- OpenCV
- MediaPipe
